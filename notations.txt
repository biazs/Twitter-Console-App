
Seção 5: Pulling Twitter Code as a DLL Library

- Why DLL (Dynamic Link Libraries) ?
Centralizar em uma biblioteca para ser reusada.

- Moving The Tweet Manager code to a Library
	Depois de clicar com botao direito na solution, Add, New project, Class Library
	Clicar (botao direito) em References (do projeto principal), Add Reference, Projects, em seguida, marcar o projeto.
	Incluir o using da nova DLL (no Program.cs)
